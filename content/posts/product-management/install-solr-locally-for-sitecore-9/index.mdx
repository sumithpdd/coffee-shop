```
choco install javaruntime -y

```

**Uninstall Solr**

There are times you may need to uninstall solr version, a quick powershell script for that

```powershell
Get-service | Where-Object {$_.Name -match "solr"} | Select-Object -ExpandProperty Name | foreach { & sc.exe DELETE $_ }
```
